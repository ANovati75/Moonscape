Download and Install the FOXSCAPE Icon Pack

Click one of the following:

View the FOXSCAPE Icon Pack for Windows readme

Download FOXSCAPE Icon Pack for Windows

After clicking the link above, a dialog box will appear. Select the Save File option and then click the OK button. After downloading the foxscape-icon-pack.zip file, you can install it by extracting the folder contained in it. Put the extracted folder into the browser\chrome folder in the folder where your Firefox program is installed. For example, on Windows XP the default location for the Firefox browser\chrome folder is C:\Program Files\Mozilla Firefox\browser\chrome . The Firefox browser\chrome folder may be in different locations on different versions of windows. Once the extracted folder is in the Firefox browser\chrome folder, all newly opened Firefox windows will have their upper left corner icons styled to match FOXSCAPE. If you would like to uninstall the icon pack, just delete the extracted folder that you placed in the Firefox browser\chrome folder. (NOTE: At present, the prior procedure is a bit more complicated than I would like it to be. I hope to improve this in a future FOXSCAPE Icon Pack release.)
